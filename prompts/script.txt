──────────────────────────────────────────────────────────────────────
STYLE GUIDE — pick style based on topic:
──────────────────────────────────────────────────────────────────────
- contrarian: tech, career, productivity — left-anchored, punchy, blue accent
- builder:    side projects, indie hacking — left-anchored, direct, blue accent
- intense:    psychology, dark truths, shocking facts — full-bleed, no margins, kinetic red
- cinematic:  money, ambition, culture — right-anchored, amber accent
- calm:       health, habits, philosophy — centered, light type weight, neutral
- analytical: data, finance, systems — centered, considered, neutral
- humorous:   fun facts, comedy, absurd observations — centered, warm amber accent;
              keywords are punchlines not statements; body text lands the joke

──────────────────────────────────────────────────────────────────────
STRICT RULES
──────────────────────────────────────────────────────────────────────

BEAT COUNT: vary between 4 and 8 depending on what the topic needs. Do NOT always use 5.

TEXT: every beat text must be 15-25 words. No exceptions.
Total words across ALL beats: 90-130 words.

KEYWORD: 2-4 words uppercase. Emotionally loaded. Scroll-stopping.
  Good: "NO ONE CARES" / "YOU'RE WRONG" / "THEY LIED"
  Bad:  "CONSISTENCY" / "PRODUCTIVITY" / "THE TRUTH"

TONE: speak as a creator with a real opinion. Use "you". Be direct. Be specific.
For humorous style: write like a stand-up bit. Keyword is setup or punchline. Body lands the absurdity straight-faced. Never explain the joke.

BANNED: "here's the thing" / "the truth is" / "most people" / "in today's world"
        "have you ever" / "let's talk about" / "follow for more"

HOOK: first sentence must create immediate tension, contradiction, or emotional recognition.

ENDING: end with a challenge, sharp question, or unresolved thought. Not a follow CTA.

──────────────────────────────────────────────────────────────────────
GLOBAL FIELDS — pick one value per field based on what fits the topic.
──────────────────────────────────────────────────────────────────────
- theme:        the visual world that matches the emotional register
- music_mood:   what would play under this script
- voice_style:  how this would be narrated
- camera_style: static=considered/calm, dynamic=intense/contrarian, handheld=raw/documentary

──────────────────────────────────────────────────────────────────────
PER-BEAT FIELDS — choose thoughtfully, NEVER randomly.
──────────────────────────────────────────────────────────────────────

⚠ VARIETY IS MANDATORY. A stiff, repetitive render is a failure.
   Across the full script you MUST:
   1. Use at least 3 different `camera` values total
   2. Use at least 2 different `pace` values total
   3. Use at least 2 different `layout` values total
   4. Use at least 2 different `background` values total
   5. Never repeat the same `camera` for 3 consecutive beats
   6. Never repeat the same `layout` for 3 consecutive beats
   If your script doesn't satisfy ALL of these, rewrite before outputting.

FIELD-BY-FIELD GUIDANCE:

- emotion:      what the audience should feel during this beat
- pace:         slow=weight/gravity, mid=normal, fast=urgency, explosive=impact
- visual_intent: the aesthetic register of this moment
- camera:       how the frame moves during this beat.
                hook=push_in, climax=snap_zoom, tension=tilt_up,
                payoff=pull_out, flip=micro_shake — but vary within reason.
- transition:   how this beat ends and the next begins.
                slam_cut for climax, dip_black for tension, fade for payoff,
                whip_pan between high-energy beats, cut as neutral default.
- background:   what sits behind the type. Rotate across beats.
                hook=glow, climax=abstract, tension=lines, insight=grid/solid.
- layout:       compositional anchor. Critical for visual rhythm:
                left   = editorial / contrarian / builder energy
                center = calm / analytical / humorous / payoff
                right  = cinematic / wealth / mirrored editorial
                full   = intense / explosive / climax bleed
                A good script alternates layouts. Two consecutive `left` is okay;
                three is monotony.
- visual_query: 2-5 word Unsplash search string for this beat's background.
                Think: what image would a human editor put behind this text?
                Match topic AND emotion. Dark, blurred, atmospheric.
                Good: "dark code terminal blur" / "person alone crowd"
                Bad:  "technology" / "business" / "abstract background"
